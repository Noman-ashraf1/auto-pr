# AutoPR AI

## Autonomous Multi-Agent Code Review & Security Audit Platform

AutoPR AI is an intelligent platform that automatically reviews GitHub Pull Requests using multiple AI agents and security scanners.

### Features

- AI Code Review
- Security Analysis
- Architecture Review
- Documentation Review
- Test Review
- Risk Scoring
- GitHub Integration
- Multi-Agent Workflow (LangGraph)

### Tech Stack

- FastAPI
- LangGraph
- PostgreSQL
- Docker
- GitHub API
- Semgrep
- Trivy
- Gitleaks
